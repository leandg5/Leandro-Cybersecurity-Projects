# Leandro Cybersecurity Projects

## Projects Overview
This repository contains various cybersecurity-related projects, including machine learning for fraud detection, WannaCry cybersecurity strategy, and more.

### Project Folders:
- **CYT-180 Home Project**: Hive and HQL queries.
- **Machine Learning Fraud Detection**: Python/PySpark for detecting fraudulent transactions.
- **CYT-100 WannaCry Cybersecurity Strategy**: A report on designing a cybersecurity strategy informed by the WannaCry ransomware.
- **Transactional Machine Learning**: Real-time data processing and cybersecurity.
